## セットアップ手順（Claude再起動時用）

### 1. 依存関係のインストール
```bash
npm install
```

### 2. 開発サーバーの起動
```bash
npm run dev
```

### 3. ビルドとテスト
```bash
npm run build
npm run lint
npm run type-check
```

## プロジェクト構造
```
salon001/
├── src/
│   ├── app/              # Next.js App Router
│   ├── components/       # React コンポーネント
│   ├── data/            # JSON データファイル
│   ├── types/           # TypeScript 型定義
│   └── utils/           # ユーティリティ関数
├── public/
│   ├── images/          # 画像ファイル
│   └── icons/           # アイコンファイル
└── docs/                # ドキュメント
```

## 設計方針

### デザインガイドライン
- **参考サイト**: Ocean Tokyo (https://www.oceantokyo.com/)
- **ブランドカラー**: 青系統
- **デザイン特徴**: モダン、清潔感、男性的
- **レスポンシブ**: モバイルファースト

### データ管理
- スタイリスト情報: `src/data/stylists.json`
- お知らせ: `src/data/news.json`
- メニュー・料金: `src/data/menu.json`
- 店舗情報: `src/data/salon.json`

### ページ構成
1. **トップページ** (`/`)
2. **スタイリスト一覧** (`/stylists`)
3. **スタイリスト詳細** (`/stylists/[id]`)
4. **スタイル写真集** (`/styles`)
5. **メニュー・料金** (`/menu`)
6. **アクセス** (`/access`)
